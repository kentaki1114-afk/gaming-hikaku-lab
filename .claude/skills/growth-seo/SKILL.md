# グロースSEOスキル

ゲーミング比較ラボ（gaming-hikaku-lab.vercel.app）のアクティブユーザーを増やすための戦略と実装手順。

---

## なぜユーザーがゼロなのか ── 根本原因の診断

| 原因 | 重要度 | 状態 |
|------|--------|------|
| ① 新ドメインサンドボックス（開設3〜6ヶ月は順位が付きにくい） | ★★★ | 時間で解決 |
| ② 記事が薄い（500字・テンプレ）→ HCUで評価されない | ★★★ | 実装改善済（本スキルで対応） |
| ③ バックリンクゼロ → ドメイン評価なし | ★★★ | 手動対応必要 |
| ④ 競合キーワード（DA50+と同じ土俵） | ★★ | キーワード戦略変更 |
| ⑤ AggregateRatingなし → 星スニペットが出ない | ★★ | 実装改善済（本スキルで対応） |
| ⑥ インデックス申請未実施 | ★★ | 手動対応必要 |

## 週次アクションチェックリスト

### 🔴 毎週やること（最重要）

1. **Search Console でインデックス手動申請**
   - https://search.google.com/search-console
   - 「URL検査」→ ランキングページ7つ＋新着記事を1つずつ「インデックス登録をリクエスト」
   - 対象: `/headsets` `/controllers` `/monitors` `/keyboards` `/mice` `/chairs` `/capture`

2. **新着記事をSNSで拡散（バックリンク代替）**
   - Twitter/Xで記事をシェア（ハッシュタグ: `#ゲーミングデバイス #PS5周辺機器`）
   - Redditの r/gaming_jp や Yahoo!知恵袋でURL付きで回答する
   - Note.comやQiitaの関連記事からリンクを貼る
   - 目標: 週3〜5件の被リンク獲得

3. **Search Console でパフォーマンスを確認**
   - 表示回数が増えているか（クリックよりまず表示回数）
   - CTRが低いページは title/description を修正
   - 平均掲載順位が20以内に来たページはコンテンツを強化

### 🟡 月次でやること

4. **ロングテールキーワードで記事を手書き追加（月2〜4本）**
   - 自動生成記事は検索流入のきっかけにならない
   - 手書き記事のターゲット例:
     - 「PS5 ヘッドセット 3Dオーディオ対応 おすすめ 2026」
     - 「ゲーミングチェア 腰痛 改善 効果 体験談」
     - 「HyperX Cloud III レビュー Switch 使い方」
     - 「Apex Legends PC コントローラー 設定」（けんたきの経験談）
   - 手書き記事は1500字以上、個人経験を必ず含める

5. **リッチリザルトテストで構造化データを検証**
   - https://search.google.com/test/rich-results
   - ランキングページ7つをテスト
   - AggregateRatingが有効かチェック（星スニペット = CTR +150%）

## コンテンツ品質改善の実装方針

### 記事生成の改善（実装済）
- 各テーマ: ~11ブロック（500字）→ ~22ブロック（1500字）
- FAQブロック追加 → FAQPage JSON-LD → FAQ検索結果スニペット
- プラットフォーム別アドバイス追加
- 「こんな人向け」読者プロファイルを追加

### 手書き記事のテンプレート
```
1. H1: [商品名] レビュー｜[使用期間]使ってわかった[カテゴリ]の[評価]
2. リード文（200字）: 誰が、何を目的に、何を得られるか
3. H2: [商品名]を選んだ理由・きっかけ（個人経験）
4. H2: 実際に使ってみた感想（良かった点3つ・気になった点2つ）
5. H2: [競合商品名]との比較
6. H2: どんな人におすすめ？
7. H2: よくある質問（3件）→ FAQPage JSON-LD
8. H2: まとめ
```

## AggregateRatingスキーマ（実装済）

ランキングページの JSON-LD に楽天レビューデータを使った AggregateRating を追加済み。
これにより Google SERP で星スニペット（⭐⭐⭐⭐⭐）が表示される可能性がある。

確認方法:
```bash
# リッチリザルトテスト（ブラウザで確認）
# https://search.google.com/test/rich-results?url=https://gaming-hikaku-lab.vercel.app/headsets
```

## バックリンク獲得戦略（手動作業）

新ドメインでの最短ルートはバックリンクの獲得。自動化不可。

### 即効性の高い施策

| 施策 | 難易度 | 期待効果 |
|------|--------|---------|
| Twitter/XでURL付きツイート | 低 | ソーシャルシグナル |
| Yahoo!知恵袋で回答＋URL | 低 | 被リンク+参照流入 |
| Reddit r/JapaneseGaming に投稿 | 低 | 英語圏被リンク |
| Note.comにサイト紹介記事を書く | 中 | 被リンク |
| Zennに「ゲーミングデバイス選び方」記事 | 中 | 被リンク+流入 |
| Steamのフォーラムで商品言及 | 低 | 参照流入 |

### 中長期

- Amazon/楽天のレビューにサイトを言及（ブランド認知）
- YouTubeにレビュー動画を作りサイトURLを概要欄に（被リンク+流入）
- X（Twitter）アカウントを育てて定期投稿
